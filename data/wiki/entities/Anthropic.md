# Anthropic

AI safety company and creator of the Claude family of models and [[Claude Code]].

## Products

- **Claude** — the underlying LLM family (Haiku, Sonnet, Opus)
- **Claude Code** — CLI-based AI coding agent
- **Claude Desktop** — desktop application
- **Claude.ai** — web interface
- **MCP** — [[Model Context Protocol]], an open standard for tool integration

## Model Lineup (as of early 2026)

- **Opus 4.6** — most capable, recommended for complex tasks. 1M context window.
- **Sonnet 4.6** — balanced speed and capability
- **Haiku 4.5** — fastest, cheapest

[[Boris Cherny]] recommends **Opus with thinking enabled** for important coding work — slower but produces fewer errors.

## Key People (mentioned in sources)

- **[[Boris Cherny]]** — creator of Claude Code
- **Barry Zhang** — Anthropic engineer, advocate for [[Skills System|skills over tools]]
- **Mahesh Murag** — Anthropic engineer, skills system development

## Strategic Direction

Sources describe Anthropic executing a coordinated platform shift from assistant to **agent operating system**, simultaneously updating browser, Slack, terminal, and mobile surfaces. Key architectural principles:

- **Open, composable layers**: MCP (open standard), Computer Use (accessibility APIs), Channels
- **Progressive disclosure** for [[Context Management|context efficiency]]
- **Skills over tools** as the primary extension mechanism

## Pricing Approach

Multiple sources suggest Anthropic is **subsidizing tokens** on Claude Code and Claude Desktop. The $200/month Claude Max plan provides approximately $5,000 worth of API usage. Potential price increases may come as the market matures.
