# GStack

A [[Claude Code]] toolkit created by Gary Tan (CEO of Y Combinator) containing nine specialized workflow [[Skills System|skills]] (slash commands).

## Installation

Requires Claude Code and Bun. Installs skills into the user's skills directory.

## Key Strengths

- Detailed planning workflows
- Automated QA processes
- Structured development flow
- Comes from a highly credible source (YC CEO)

## Reception

One source committed to a **30-day trial** of GStack, replacing the Superpowers plugin as their main Claude Code workflow tool. Key factors: the detailed planning, automated QA, and the credential of coming from YC's CEO.
