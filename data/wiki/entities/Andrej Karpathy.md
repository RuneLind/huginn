# Andrej Karpathy

AI researcher, educator, and builder. One of the most influential voices on the practical future of AI coding and agents.

## Background

- Taught Stanford's most influential deep learning course
- Co-founded [[OpenAI]]
- Led Tesla's Autopilot vision team
- Launched **Eureka Labs** — AI education company
- Formerly one of the most senior AI researchers in the industry

## AutoResearch

Karpathy's most impactful recent project: an open-source pipeline that lets AI agents **run experiments autonomously**.

- Give it a research question and a `train.py` script
- The agent designs experiments, runs them, analyzes results, writes reports
- Acts like a "super nerd robot intern" — tireless, methodical
- Sparked widespread adoption and inspired the [[Skills System|self-improving skills]] pattern in [[Claude Code]]
- Open-sourced on GitHub; quickly became one of the most-starred AI repos

## 10 Principles for AI Agents

A widely-cited framework:

1. Keep agents simple (minimal tools, clear loops)
2. Use natural language as the interface
3. Give agents access to the terminal
4. Let agents fail and retry
5. Build in verification
6. Replace bespoke apps with agent-driven API glue (his "Dobby" home automation assistant connects everything via WhatsApp)
7. Make agents persistent (long-running loops, not one-shot)
8. Embrace the "loopy era" — agents running continuously, not just responding
9. Let agents accumulate knowledge
10. Start small, iterate fast

## On the Future of Coding

- Coined **Jevons Paradox** applied to coding: as AI makes coding cheaper, the total demand for code explodes rather than shrinks
- Argues **coding jobs aren't going anywhere** — they're transforming. Engineers will manage agents, not write every line
- Predicts a shift from "writing code" to "managing code agents"
- Built **"Dobby the Elf Claw"** — a persistent home automation agent controlling Sonos, lights, HVAC, shades, pool, and security via WhatsApp
