# Sources — RAG

Catalog of 7 YouTube video summaries about RAG (Retrieval-Augmented Generation) and knowledge systems. Raw files in `raw/ai/rag/`.

## RAG Evolution

- **Introducing RAG 2.0: Agentic RAG + Knowledge Graphs (FREE Template)** — next-generation RAG with agents and knowledge graphs
- **RAG is Dead: Introducing Agentic File Exploration** — argument that traditional RAG is being replaced by agentic approaches (2 versions of this source)
- **10x Your RAG System With Google's New Model and Claude Code** — Google model + [[Claude Code]] for enhanced RAG
- **Google's New Model + Claude Code Just Changed RAG Forever** — overlapping coverage of Google + Claude RAG

## Specialized Knowledge Systems

- **Claude Code + LightRAG = UNSTOPPABLE** — LightRAG integration with Claude Code
- **Folders Are Broken, So I Built Heaper** — alternative to folder-based file organization

## Key Theme

The consensus across sources: traditional RAG (chunk → embed → retrieve → generate) is giving way to **agentic approaches** where the AI actively explores and navigates documents rather than passively retrieving chunks. Knowledge graphs add relational context that flat vector search misses.
