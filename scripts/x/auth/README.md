# X/Twitter Authentication

This directory contains authentication files for X/Twitter access.

## Required File

**x_auth.json** - Browser session state from Playwright
- Generated by running `auth_setup.py`
- Contains cookies and authentication state
- **DO NOT commit to git** (already in .gitignore)

## Setup

1. Run: `uv run scripts/x/auth_setup.py`
2. Log in via the browser window that opens (handles 2FA, captcha, etc.)
3. Wait for redirect to x.com/home
4. The session will be saved to `x_auth.json`
5. Re-run when session expires (typically every few weeks)
