# X/Twitter Authentication

This directory contains authentication files for X/Twitter access.

## Required File

**x_auth.json** - Cookies extracted from your real browser
- Generated by running `auth_setup.py`
- Contains `auth_token` and `ct0` cookies
- **DO NOT commit to git** (already in .gitignore)

## Setup

1. Open Chrome and go to x.com (make sure you're logged in)
2. Run: `uv run scripts/x/auth_setup.py`
3. Open DevTools (Cmd+Option+I) → Application → Cookies → https://x.com
4. Copy `auth_token` and `ct0` cookie values when prompted
5. The script verifies your cookies work before saving

Re-run when session expires (typically every few weeks).
